\chapter{Use Case Specifications}
\label{chap:use-case-specifications}

This chapter specifies the major use cases supported by the EcoTrace
Intelligent Electronic Waste Recovery and Circular Economy Management
System.

Each use case describes how users interact with the system to achieve a
specific objective. The use cases provide detailed operational
descriptions that support software design, implementation, testing, and
validation.

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%

\section{Introduction}

The EcoTrace platform supports multiple categories of users, each with
different responsibilities and system privileges.

The use cases presented in this chapter describe the interactions
between users and the software system during normal and exceptional
operating conditions.
